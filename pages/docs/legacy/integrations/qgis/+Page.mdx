# QGIS integration

Macrostrat can be used as a data source in QGIS. We provide QGIS layer files
for the following harmonized map layers:

- [Carto](/docs/integrations/qgis/layers/macrostrat-carto.qlr)
- [Carto slim](/docs/integrations/qgis/layers/macrostrat-carto-slim.qlr)
